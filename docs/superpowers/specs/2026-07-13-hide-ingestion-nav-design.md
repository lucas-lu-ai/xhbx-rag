# 隐藏文档入库导航按钮设计

## 目标

暂时隐藏 Web 工作台导航中的“文档入库”按钮，同时完整保留文档入库页面、URL 路由、前端组件、后端接口及已有数据处理能力。

## 设计

- `WorkspaceNav` 不再渲染“文档入库”按钮，仅保留“知识问答”按钮。
- 不修改 `WorkspaceLocation`、路由解析与导航状态，因此已有文档入库 URL 仍可直接访问。
- 不删除或禁用任何文档入库组件、轮询逻辑、上传逻辑或后端路由。
- 文档入库页面内同样使用共享导航，因此该页面也不显示“文档入库”按钮。
- 移除按钮不再使用的 `FileUp` 图标导入。

## 测试

- 增加共享导航测试，断言“知识问答”存在且“文档入库”按钮不存在。
- 将依赖点击“文档入库”按钮进入页面的现有测试改为通过路由状态进入，保留文档入库功能覆盖。
- 运行相关测试、完整前端测试及生产构建。
